# Lesson 013: Transportation & Directions - Exercises

## Round 1 (Translation)
1. **Prompt:** I like only movies.
   - **User:** 영화를만 좋아해요 -> **영화만** (Drop object particle)
2. **Prompt:** The bus arrives in about 5 minutes.
   - **User:** 오분 정도에 버스가 도착해요 -> **5분 정도 후에** (or 5분 정도면)
3. **Prompt:** Among fruits, I like bread.
   - **User:** 과일 중에서 빵을 좋아해요 (Correct!)
4. **Prompt:** Please wait only at the front door.
   - **User:** 앞문에만 기다려세요 -> **앞문에서만 기다리세요**
5. **Prompt:** The bus fare is about 1,200 won.
   - **User:** 버스 요금이 가격은 찬이백 원 정도 해요 -> **요금이 천이백 원 정도 예요**

## Round 2 (Spatial & Particles)
1. **User:** 버스가 앞문으로 와요 (Correct!)
2. **User:** 뒷몬으로 내리세요 -> **뒷문으로** (Spelling)
3. **User:** 싼 빵만 머거요 -> **먹어요**
4. **User:** 교통 충에서 지하철을 좋아해요 -> **중에서**
5. **User:** 십분 정도 기대려요 -> **기다려요**

## Active Production
1. **Prompt:** Among transportation, what do you like?
   - **User:** 빠른 있서 차를 겨통 정에서 젛아해요
   - **Correction:** **빨라서 교통 중에서** 차를 **좋아해요**.
2. **Prompt:** How long do you wait?
   - **User:** 부통 10분 점도 기다려요 -> **보통 10분 정도**
3. **Prompt:** Where only can you take the bus?
   - **User:** 정료장에서 버스를 탈 수 있어요 -> **정류장에서만**

## Active Writing
1. **User:** 집에서만 기다려료 -> **기다리세요(기다려요)**
2. **User:** 김치는 음식 중에서 좋아애요 -> **음식 중에서 김치를 좋아해요**
3. **User:** 1 시간 정도 갈려요 -> **걸려요**
