# Lesson 014 Review

## Performance
- **Score:** 8/10
- **Strengths:** Strong use of the comparison particle `-보다`. Correct application of `가장/제일` for superlatives. Successfully integrated nominalization and modifiers into comparisons.
- **Weaknesses:** Spelling of common words (지하철, 오늘, 중, 초콜릿). ㄹ-irregular modifier rules (만든, 사는).

## Vocabulary Learned
- **보다:** Than (comparison particle).
- **더:** More (adverb).
- **가장 / 제일:** The most / The best.
- **훨씬:** Much / Way more.
- **키 (키가 커요):** Height (Taller).
- **어제 / 오늘 / 내일:** Yesterday / Today / Tomorrow.
- **여름 / 봄 / 가을 / 겨울:** Summer / Spring / Autumn / Winter.
- **달다 / 맵다 / 시다 / 쓰다:** Sweet / Spicy / Sour / Bitter.

## Grammar Covered
- **The Comparison Formula:** A가 B보다 (더) [Adj/Verb].
- **Superlatives:** [Category] 중에서 [Item]이 가장 [Adj/Verb].
- **Adverb Usage:** Using `더` and `가장` with verbs and adverbs (더 빠르게).
