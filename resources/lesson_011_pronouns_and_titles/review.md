# Lesson 011 Review

## Performance
- **Score:** 9/10
- **Strengths:** Excellent understanding of relational pronouns and hierarchy. Successfully avoided "you" by using titles/names. Correctly distinguished between `이에요/예요` (definition) and `있어요` (location).
- **Weaknesses:** Minor vowel spelling slips (어디, 선생님, 이름). Early confusion with the verb "to be" conjugation.

## Vocabulary Learned
- **저 / 나:** I (Formal / Casual)
- **저희 / 우리:** We (Formal / Casual)
- **제 / 내:** My (Formal / Casual)
- **씨 / 님:** Honorific suffixes for names/titles.
- **선생님:** Teacher / Doctor (honorific).
- **형 / 오빠 / 누나 / 언니:** Older sibling terms used as "you".
- **이것 / 그것 / 저것:** This / That (near you) / That (far).
- **얼마:** How much.
- **이름:** Name.

## Grammar Covered
- **The Verb "To Be":** N + 이에요 (cons.) / 예요 (vowel).
- **The Negative "To Be":** N + 이/가 아니에요.
- **Possessive Contractions:** 저의 -> 제, 나의 -> 내.
- **The "You" Trap:** Strategies for avoiding second-person pronouns in Korean.
- **Communal "Our":** The use of 우리 for personal items (우리 집, 우리 엄마).
